# 法学笔记整理 Skill

## 概述

本 Skill 用于将法学教材整理为高质量 Markdown 笔记。

**核心目标**：在进行"文本压缩"的同时，保持"信息丰富"，做出"具有高信息密度"的笔记。

本 Skill 只负责一件事：**从原始教材 Markdown → 整理后的笔记 Markdown**。

---

## 核心理念

### 1. 无损压缩理念

**定义**：笔记整理不是简单的删减，而是在保留所有关键信息的前提下进行压缩。

**原则**：
- **禁止**：教材讲好几页的内容，笔记几句话略过
- **要求**：读者读笔记的理解程度 ≈ 读教材的理解程度
- **方法**：
  - 用表格压缩对比性内容（如学说对比）
  - 用列表压缩并列性内容
  - 用结构化语言压缩叙述性内容

### 2. 信息可视化理念

**定义**：能用表格、图表呈现的内容，不用纯文字叙述。

**必须使用表格的场景**：
- 学说对比（人物、核心主张、评价）
- 分类列举（三要素、三原则、三阶段）
- 时间线/发展历程
- 法条对比（新旧法对比）

**禁止出现的情形**：
- 大段的文字堆叠（无表格、无序号、无加粗）
- 整段照搬原文内容而不加整理
- 可以结构化的信息用大段文字输出

### 3. 信息密度最大化

**目标**：每一行文字都应承载有效信息。

**方法**：
- 删除过渡性、铺垫性语句
- 删除重复性表述
- 删除与核心知识无关的背景信息
- 合并相似要点
- 使用精准、凝练的法律术语

**禁止**：凑字数、堆叠无用信息

### 4. 避免重复性表述

**处理原则**：
- 概念首次出现时完整定义
- 后续提及用简称或指代
- 同一要点不在多处重复展开
- 案例说理与概念说明不重复

### 5. 禁止篡改、编造内容

**红线**：不得杜撰、虚构、编造。

**法条要求**：
- 必须来自教材原文或权威法律数据库
- 必须保留准确的条文编号
- 必要时可联网搜索核实

**案例要求**：
- 必须来自教材原文或权威判例
- 必须保留案名、关键事实、判决要旨
- 不得自行编造案例名称或案情

## 一、格式标准（强制性）

### 1.1 标题层级体系

**⚠️ 绝对禁止：以"章"作为一级标题**

#### ✅ 正确格式：

```markdown
# 第一节 节标题

**本节评估**：教材约X页，知识点密度[高/中/低]，考试重要度[高/中/低] → **[A/B/C级]**

---

## 一、一级标题

### （一）二级标题

#### 1. 三级标题（极少使用）

**核心概念**：具体内容

**重要说明**：补充解释
```

#### ❌ 绝对禁止：

```markdown
# 第四章 章标题          ← 禁止！不用输出章标题
## 第一节 节标题         ← 禁止！一级必须是#
```

#### 层级对应关系：

| 级级 | 格式 | 说明 |
|:----:|:-----|:-----|
| 一级 | `# 第一节 节标题` | 只能是"节"，禁止用"章" |
| 二级 | `## 一、标题内容` | 中文数字 |
| 三级 | `### （一）标题内容` | 括号中文数字 |
| 四级 | `#### 1. 标题内容` | 极少使用 |

> 完整的格式示例和更多细节见 **[references/format-standards.md](references/format-standards.md)**
> 整理后的笔记成品示例见 **[references/example.md](references/example.md)**
> 详细的压缩技巧和逐类型处理指南见 **[references/compression-guide.md](references/compression-guide.md)**

### 1.2 固定模块标签

| 标签 | 使用场景 | 示例 |
|:-----|:---------|:-----|
| **核心概念** | 概念定义 | 涉外 = 涉及外国因素 |
| **重要说明** | 关键限定 / 补充解释 | 上述三要素只要其一涉外即可 |
| **立法进步** | 法律演变评价 | 新增"经常居所地"标准 |
| **学术争议** | 不同学说观点 | 肯定说 vs 反对说 |
| **历史意义** | 历史地位评价 | 被誉为"近代国际私法之父" |
| **理论缺陷** | 学说局限性 | 循环论证问题 |
| **方法论贡献** | 方法论创新 | 采用实证方法 |

### 1.3 表格格式

```markdown
| 列1 | 列2 | 列3 |
|:-----|:-----|:-----|
| 内容 | 内容 | 内容 |
```

### 1.4 强调与引用

| 格式 | 使用场景 | 示例 |
|:-----|:---------|:-----|
| `**加粗**` | 核心概念、关键术语 | **法律关系本座说** |
| `> 引用块` | 法条原文、经典表述 | > "诸化外人，同类自相犯者，各依本俗法" |
| `---` | 分隔大段内容 | --- |

### 1.5 英文翻译保留规则

| 类型 | 处理方式 | 示例 |
|:-----|:---------|:-----|
| 概念术语 | 括号内保留 | 万民法(jus gentium) |
| 学说名称 | 括号内保留 | 法则区别说(statuta theory) |
| 人名 | 括号内保留拼写 | 萨维尼(Savigny) |
| 法学术语 | 括号内保留拉丁文 | 场所支配行为(locus regit actum) |
| 普通名词 | 可省略 | "方法"、"原则"等 |

## 二、内容评估与篇幅分配

### 2.1 章节等级划分

每节开头需附上等级评估：

```markdown
**本节评估**：教材约X页，知识点密度[高/中/低]，考试重要度[高/中/低] → **[A/B/C级]**

| 等级 | 评分范围 | 笔记篇幅 | 策略 |
|:-----|:--------:|:---------|:-----|
| **A级** | 80-100 | 教材×80-120% | 详细展开，几乎不删 |
| **B级** | 50-79 | 教材×50-80% | 平衡处理，适度压缩 |
| **C级** | 0-49 | 教材×30-50% | 高度概括，只留核心 |

### 2.2 压缩率计算公式

```
压缩率 = (原文件字符数 - 笔记文件字符数) / 原文件字符数 × 100%
```

**字符统计**：去除空格和换行后的纯文本字符。

### 2.3 强制压缩率标准

| 压缩率 | 状态 | 处理 |
|:------:|:----:|:----:|
| <35% | 偏低 | ✅ 可接受，内容较详细 |
| **35-65%** | **正常** | ✅ **理想范围** |
| 65-75% | 偏高 | ⚠️ 需检查，可能存在过度压缩 |
| >75% | **禁止** | ❌ **必须重做** |

**绝对禁止**：压缩率 >75% 的章节直接提交。

> 详细的压缩技巧和逐类型处理指南见 **[references/compression-guide.md](references/compression-guide.md)**

## 三、详略取舍标准

### 3.1 必须详细保留（压缩率 <30%）

| 内容类型 | 保留要求 |
|:---------|:---------|
| 核心概念定义 | 完整保留，准确表述 |
| 学说核心主张 | 保留完整论点、代表人物、标志性表述 |
| 法典条文原文 | 引用块完整引用，保留条文编号 |
| 典型案例 | 保留案名、核心事实、判决要旨 |
| 学说对比 | 以表格形式保留各家观点差异 |
| 学者评价 | 保留权威性评价 |

### 3.2 适当压缩（30-55%）

| 内容类型 | 处理方式 |
|:---------|:---------|
| 历史背景 | 概括关键转折点和因果关系 |
| 论证过程 | 压缩为逻辑链条或要点 |
| 脚注引用 | 保留核心文献，一般性引用可省略 |
| 案例细节 | 保留关键要素，省略过程性描述 |
| 重复性表述 | 合并或省略 |

### 3.3 可大幅压缩或省略（>55%）

| 内容类型 | 处理方式 |
|:---------|:---------|
| 冗长的历史叙述 | 时间线或一句话概括 |
| 过细的脚注 | 仅保留页码和核心观点 |
| 重复举例 | 首次完整，后续提及即可 |
| 过渡性语句 | 删除 |
| 教材中的自谦/客套表述 | 删除 |

## 四、文件路径与命名规范

### 4.1 输入输出目录结构

**输入目录**（原始教材，已按小节切分的 md 文件）：

```
~/书名/
├── 第01章/
│   ├── 第一节 xxx.md
│   ├── 第二节 xxx.md
│   └── 第三节 xxx.md
├── 第02章/
│   └── ...
```

**输出目录**（整理后的笔记）：

```
~/书名_笔记/
├── 第01章/
│   ├── 第一节 xxx.md        ← 整理后
│   ├── 第二节 xxx.md        ← 整理后
│   └── 第三节 xxx.md        ← 整理后
├── 第02章/
│   └── ...
```

**路径规则**：
- 输出根文件夹 = 输入文件夹名 + `_笔记`
- 保持原有章节结构和文件名不变
- 每个小节 md 文件单独整理，一一对应

### 4.2 大章节处理（A级章节细分）

**适用条件**：单节原始文件过大（建议超过 500 行或单个文件整理时上下文不足），且等级评估为 A 级。

**细分规则**：
- 按"一、二、三..."二级标题切分
- 先将原始小节拆分为多个子部分分别整理
- 最后合并为一个完整的输出文件

**处理流程**：

```
1. 读取原始小节 md 文件
2. 识别"一、二、三..."二级标题位置
3. 将各部分内容分别整理
4. 按顺序合并为一个完整的笔记 md 文件
5. 写入输出路径 ~/书名_笔记/章名/小节名.md
```

## 五、工作流程

### Step 1: 确定输入输出路径

根据用户指定的书籍名称，确定：
- 输入根目录：`~/书名/`
- 输出根目录：`~/书名_笔记/`

扫描输入目录，获取所有章节和小节文件列表。

### Step 2: 读取 & 评估每个小节

逐个读取每个 `.md` 文件，做快速扫描：

1. **统计规模**：字符数、大致行数
2. **识别结构**：找到已有的标题层级、法条引用、案例段落
3. **评估等级**：判断本节的知识点和考试重要度 → A/B/C 级

### Step 3: 按规范整理

对照上述格式标准（第一部分）和详略取舍标准（第三部分），逐节整理。

**核心操作顺序**：
1. 通读整节，理解逻辑脉络
2. 重构标题层级（确保以 `# 第X节` 开头）
3. 按等级决定详略：A 级详细展开，C 级高度概括
4. 学说对比 → 表格；法条原文 → 引用块；概念定义 → 加粗标签
5. 删除过渡句、重复表述、自谦客套话
6. 检查压缩率是否在 35-65% 范围内（不要写入笔记）

### Step 4: 写入输出

将整理完成的笔记写入对应输出路径 `~/书名_笔记/章名/小节名.md`。

### Step 5: 自检

完成每节整理后，对照质量检查清单确认。

## 六、质量检查清单（每节必做）

- [ ] **压缩率** 35-65%（禁止 >75%）
- [ ] **标题层级正确**：一级=#节 → 二级=##一、 → 三级=###（一）
- [ ] **无"章"标题**：一级标题没有使用"第X章"
- [ ] **法条用引用块**：法条原文以 `>` 引用块呈现
- [ ] **学说对比用表格**：多学说对比使用了表格格式
- [ ] **核心概念有标签**：使用了 **核心概念** / **重要说明** 等标签
- [ ] **英文翻译保留**：人名、术语、学说名称括号内有英文/拉丁文
- [ ] **无杜撰内容**：所有法条、案例、数据均来自原文件
- [ ] **本节评估标注**：文件开头有 A/B/C 级评估

---

## 七、绝对禁止事项

1. **禁止杜撰**：不得编造法条、案例名称、案情、数据
2. **禁止篡改原意**：不得修改教材的核心观点和论证方向
3. **禁止以"章"为一级标题**：一级标题必须是"第X节"
4. **禁止过度压缩**：压缩率 >75% 必须重做
5. **禁止丢失原始文件**：原始输入文件必须保留不动
6. **禁止基于旧版本修改**：始终从原始输入文件重新整理

## 核心口诀

**删冗余不删要点，用表格不用长文，查压缩率防偷懒。**
